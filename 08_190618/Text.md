# rosbag
## カメラの使用方法
* 参考「uvc_camera」（[English](http://wiki.ros.org/uvc_camera)）
```
[Terminal 1]
$ roscore

[Terminal 2]
$ rosrun uvc_camera uvc_camera_node # もしくは以下
$ rosrun uvc_camera uvc_camera_node _device:=/dev/video1

[Terminal 3]
$ rviz
```
